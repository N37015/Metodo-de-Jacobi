El método de Jacobi es un algoritmo iterativo utilizado para encontrar aproximaciones sucesivas de los autovalores y autovectores de una matriz real simétrica. Este método es especialmente útil en problemas de álgebra lineal y análisis numérico. Dado un sistema de ecuaciones lineales de la forma Ax=b, donde A es una matriz cuadrada, X es el vector de incongnitas y B es el vector del lado derecho, el metodo de jacobi se utiliza para encontrar una solucion aprocximada.
